# Personal Task Manager

A full-stack Task Manager application built with React, Node.js, Express, and JSON file storage.

## Features

### Backend

- REST API with Express
- JSON file storage
- CRUD Operations
- Task filtering

### Frontend

- React + Vite
- Axios integration
- Add Task
- Edit Task
- Delete Task
- Toggle Task Status
- Search Tasks
- Filter Tasks
- Statistics Dashboard
- Responsive Design

## Tech Stack

### Frontend

- React
- Vite
- Axios

### Backend

- Node.js
- Express

### Storage

- JSON File

## Installation

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
